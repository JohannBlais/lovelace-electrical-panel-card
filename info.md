# Electrical Panel Card

Interactive one-line electrical panel diagram for Home Assistant Lovelace.

- Live power readings on phases, RCDs, circuits and individual loads
- Smart-plug toggle controls embedded in the diagram
- Per-circuit / per-zone breakdown with floor and room labels
- 100 % YAML configuration — adapt it to any panel layout

Lightweight (single SVG, no external requests, uses the standard `hass` object — no token required).
